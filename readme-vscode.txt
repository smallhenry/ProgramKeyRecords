VSCode 熱鍵======================================================
alt+shift + F  	:排版處理
ctrl+ enter		:另一視窗開啟
ctrl + P 		:模糊搜尋檔名
ctrl + w 		:關閉檔案
ctrl + \ 		:切分現有視窗
ctrl + 1~3		:切換第幾各視窗
ctrl+shift+D 	:開始建立組態檔(Debug)
ctrl+shift+M	:顯示程式錯誤位置
ctrl+shift+P	:呼叫 task run，VSCode 會自動撈出 gulpfile 內的 Task 讓開發者選擇執行。
